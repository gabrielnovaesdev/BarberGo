# 💈 BarberGo

O BarberGo é um projeto desenvolvido com foco em aprendizado prático e evolução técnica no desenvolvimento de software. A aplicação foi criada utilizando inteligência artificial como ferramenta de apoio durante o processo de construção, explorando desde a geração de ideias até a estruturação de componentes, organização de código e refinamento da interface.

Mais do que apenas um projeto funcional, o BarberGo representa um ambiente de estudos voltado para o aprimoramento de habilidades em desenvolvimento web, lógica de programação, UI/UX e utilização estratégica de IA no fluxo de desenvolvimento.

## 🎯 Objetivo do Projeto

Este repositório foi criado com o propósito de:

* Praticar desenvolvimento front-end e estruturação de aplicações;
* Aprimorar conhecimentos em tecnologias web modernas;
* Explorar boas práticas de organização e escrita de código;
* Desenvolver habilidades em engenharia de prompts;
* Entender como utilizar inteligência artificial como ferramenta de produtividade e aprendizado.

## 🧠 Uso de Inteligência Artificial

Parte do desenvolvimento contou com apoio de ferramentas de IA para:

* Geração e refinamento de código;
* Estruturação de componentes;
* Sugestões de interface;
* Correção e melhoria de lógica;
* Aprimoramento da documentação;
* Criação de prompts mais eficientes e específicos.

O projeto também serviu como prática para aprender a elaborar prompts claros e objetivos, obtendo respostas mais consistentes e úteis durante o desenvolvimento.

## 🚀 Aprendizados

Durante a construção do BarberGo foram trabalhados conceitos como:

* Organização de projetos;
* Estruturação de interfaces;
* Componentização;
* Responsividade;
* Versionamento com Git e GitHub;
* Escrita de documentação técnica;
* Fluxo de desenvolvimento auxiliado por IA.

## 📚 Finalidade

Este projeto possui finalidade educacional e faz parte da minha jornada de aprendizado em tecnologia e desenvolvimento de software.
